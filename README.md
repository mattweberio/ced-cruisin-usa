# CED Cruisin' USA

Arcade driving game for Cedric's retirement send-off.

## Play

Open `homura.it.com/sandbox/ced-cruisin/` in a browser. Press SPACE to start.

## Controls

- **Left / Right** or **A / D** — Steer
- **Up** or **W** — Boost (1.5s burst, 4s cooldown)
